# NFT Mint Tracker (Web3)

This project tracks NFT mint activity and shows simple NFT information.

It is made for beginners who want to learn how NFTs and smart contracts work.

## What this project does

- Simulates NFT minting
- Stores minted NFT data
- Shows NFT owner info
- Displays mint history

## Why this project is useful

You will learn:
- How NFT minting works
- How smart contracts store data
- How blockchain events are tracked
- How to organize a Web3 project

## Tech used

- JavaScript
- Node.js
- NFT concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `mint.js` → NFT mint logic  
- `nft.js` → NFT data store  
- `owner.js` → Owner validation  
- `history.js` → Mint history  
- `config.js` → Settings  
- `utils.js` → Helpers  
- `logger.js` → Logs  
- `sample.env` → Config example  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use real smart contracts or wallets.
